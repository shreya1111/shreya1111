## ⟶ Featured Projects



### 🩺 MedNexus AI
**Production-Grade AI Healthcare Assistant**

An AI healthcare assistant combining retrieval-augmented generation with a persistent vector knowledge base to deliver grounded, context-aware medical information through a FastAPI backend.

**Architecture** — FastAPI service layer → LangChain orchestration → ChromaDB vector store → Gemini API for generation, with retrieval grounding to reduce hallucination.

**Key Features**
- Retrieval-augmented question answering over medical knowledge
- Persistent vector storage with ChromaDB
- LangChain-orchestrated LLM pipeline
- FastAPI backend designed for real request/response workloads

**Tech Stack**
`Python` `FastAPI` `LangChain` `ChromaDB` `Gemini API` `LLMs` `RAG`

**Future Improvements** — Add conversation memory, response citation grounding, and a lightweight evaluation harness for answer quality.
